# Financial-Risk-Intelligence-RAG-System
A Python-based application to analyze and compare financial risks from company 10-K reports using Retrieval-Augmented Generation (RAG).

## Features
- Extracts and processes financial documents
- Performs risk comparison using LLM-based approach
- Streamlit dashboard for visualization

## Tech Stack
- Python
- Streamlit
- FastAPI (prototype)
- FAISS (for retrieval)

## Note
This is a prototype project demonstrating data processing, retrieval, and application development.
